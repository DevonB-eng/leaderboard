# leaderboard

**Idea** - app that monitors a group's screentime and posts a screentime leaderboard at the end of each day. 

**Implementation** 
- flutter allows for development of both ios and android
- firestone database allows for real time updating and group creation

**Features**
- unlimitied sized groups
- real time screentime leaderboard displayed for each group (only displaying "bad" apps: social medias & such)
- ability to add or remove apps to "bad apps" list per group decision
